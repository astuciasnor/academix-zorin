Guia Completo: Configurando o GRUB (Dual Boot)

Introdução

O GRUB (Grand Unified Bootloader) é o menu que aparece quando você liga o computador, permitindo escolher entre Zorin OS e Windows. Este guia mostra como definir o Windows como a opção padrão, para que ele inicie automaticamente se você não tocar em nada.

Atenção: A edição de arquivos de boot é uma operação sensível. Siga os passos com atenção. Um erro de digitação pode afetar a inicialização do sistema. Faça por sua conta e risco.

Passo 1: Identificar a Entrada do Windows

Primeiro, precisamos saber o nome exato que o GRUB usa para o Windows. Abra o terminal e execute:

grep --color=never "menuentry '" /boot/grub/grub.cfg

A saída listará todas as entradas do menu. Procure pela linha do Windows e **copie o texto exato que está entre as aspas simples (' ').**

Exemplo de saída:
menuentry 'Zorin OS 17.1' ...
menuentry 'Windows Boot Manager (on /dev/nvme0n1p1)' ...
No caso acima, o texto a ser copiado é: Windows Boot Manager (on /dev/nvme0n1p1)

Passo 2: Editar o Arquivo de Configuração do GRUB

Agora, vamos editar o arquivo de configuração principal do GRUB com o editor de texto `nano`:

sudo nano /etc/default/grub

Dentro do editor, localize a linha que diz GRUB_DEFAULT=0. Apague o 0 e cole o texto que você copiou, agora entre aspas duplas (" "). A linha deve ficar assim:

GRUB_DEFAULT="Windows Boot Manager (on /dev/nvme0n1p1)"

Passo 3: Ajustar o Tempo de Espera (Opcional)

No mesmo arquivo, você pode alterar o tempo que o menu fica visível. Encontre a linha GRUB_TIMEOUT=10 e altere o valor `10` para o número de segundos que desejar (ex: `5` para 5 segundos). Para que o menu sempre espere sua escolha (não inicie automaticamente), mude para GRUB_TIMEOUT=-1.

Passo 4: Salvar e Aplicar as Mudanças

Para salvar o arquivo no editor `nano`, pressione Ctrl+O, depois Enter para confirmar, e então Ctrl+X` para sair.

De volta ao terminal, execute o comando final para que o sistema aplique suas novas configurações:

sudo update-grub

Passo 5: Reiniciar e Verificar

Pronto! Na próxima vez que reiniciar o computador, o Windows estará pré-selecionado no menu do GRUB.
