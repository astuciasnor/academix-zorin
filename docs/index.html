Manual do Usuário Academix-Zorin

Bem-vindo ao guia de pós-instalação do Academix-Zorin! Se você está lendo isto, significa que executou com sucesso nosso script de automação e agora possui uma poderosa estação de trabalho pronta para suas atividades acadêmicas e científicas.

O que este manual cobre?
  • Dicas iniciais de uso para os principais programas instalados.
  • Instruções para configurações comuns, como o GRUB para dual boot.
  • Links e recursos para você se aprofundar em cada ferramenta.

Use o menu à esquerda para navegar rapidamente até a seção do programa sobre o qual você deseja saber mais.

Primeiros Passos

O instalador automatizado do Academix-Zorin preparou seu sistema e instalou um conjunto completo de softwares. A maneira de iniciar o processo foi simples:

  1. Clonar o repositório do GitHub: 
    git clone https://github.com/astuciasnor/academix-zorin.git
  2. Entrar no diretório: 
    cd academix-zorin
  3. Executar o script mestre: 
    ./academix-zorin.sh

Após a conclusão e uma reinicialização do sistema, todos os programas listados neste manual estarão disponíveis no seu menu de aplicativos.

WPS Office

O WPS Office foi instalado e configurado com a tradução para Português (Brasil). Ele é a sua suíte de escritório principal, compatível com os formatos do Microsoft Office.

Dica: A integração com o Zotero já foi configurada. Procure pela aba "Zotero" no WPS Writer para começar a adicionar citações e bibliografias aos seus documentos.

Zotero

Seu gerenciador de referências, Zotero, foi instalado via Flatpak, garantindo atualizações fáceis e estabilidade. Para mantê-lo e outros aplicativos Flatpak atualizados, use o comando:

flatpak update
