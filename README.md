# academix-zorin

**academix-zorin** é um instalador gráfico e automatizado para deixar o Zorin OS pronto para uso acadêmico, científico e profissional, com ferramentas configuradas em português, sem complicações.

## Objetivo

Fornecer um ambiente Linux funcional para estudantes, pesquisadores e profissionais com foco em:

- Escrita acadêmica e gerência de referências (WPS Office, Zotero)
- Ciência de dados (R, RStudio, Python, Quarto)
- Visualização e edição de imagens (Pinta, Flameshot)
- Navegação e produtividade (Chrome, OnlyOffice)
- Tradução da interface e personalização de atalhos
